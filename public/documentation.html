Movie API

This API is the server-side component of a movies web applicaiton that will allow users to access information about different movies, directors, and genres.
Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Endpoints

Request URL HTTP Method Request Data Format Response Data Format
List All Movies /movies GET None A JSON object holding a complete list of movies in the database.
Return Data About a Single Movie /moves/[Title] GET None A JSON object holding data about a single movie. Example:

{ Image URL: "#", Title: "Howl's Moving Castle", Genre: { "Romance", "Fantasy" } Description: "description text here", Director: "Hayao Miyazaki", Featured: "Yes" }

Return Data About a Genre /movies/[Genre] GET None A JSON object holding data about a genre. Example:

{ Genre: “Fantasy”, Description: “description text here”, Titles: { “Howl’s Moving Castle”, “Castle in the Sky” } }

Return Data About a Director /movies/[Name] GET None A JSON object holding data about a director. Example:

{ Name: “Hayao Miyazaki”, Born: “1941”, Died: “-“, Bio: “biography text here”, Titles Directed: { “Howl’s Moving Castle”, “Castle in the Sky” } }

Allow New Users to Register /account POST A JSON object holding data about the user registering a new account. Example:

{ Name: “Kelly”, Username: “kellysmovies” }

A text message indicating that the user has successfully registered.
Allow Users to Update Their Username /account/[username] PATCH None A text message indicating that the user has successfully updated their username.

Allow Users to Remove a Movie from their List /favorites DELETE None A text message indicating that the user has successfully deleted a movie from their list of favorites.
Allow Users to Deregister /account DELETE None A text message indicating that the user has successfully deregistered their account.
